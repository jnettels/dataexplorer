DataExplorer

The DataExplorer displays your data in an interactive way and helps to find correlations. It is powered by Python and Bokeh.

Usage of the application

Scatters

  • This tab shows the plots of your data
  • You can filter your data by the classes that were recognized in your data
  • One classification can be selected to be used for the coloured legend
  • Experiment with the tools in the toolbar on the left to pan and zoom the view
  • The selection tools help to focus on specific scatter points
  • Hold Shift to make multiple selections

Data Table

  • This tab shows the raw data in a table view
  • Selections made in Scatters will be highlighted here, and vice versa
  • Use Shift and Ctrl for selections of multiple rows

Correlation

  • This tab shows a colour coded matrix of the Pearson correlation coefficients
  • Hovering over the tiles shows the numeric values of the coefficients
  • The selected tiles represent the plots shown in Scatters
  • You can change the selection with the Tap tool. Hold Shift to make multiple selections
  • Once you return to Scatters, the requested plots will be generated

Settings

  • This tab features several settings for the application
  • Upload new data and download the currently selected data
  • Change properties of the plots
  • Select which plots to show
  • Select which value columns and classifications are used for the plots. Increasing the number of value columns increases the potential number of plots with the power of two, and too many plots will crash the application. For this reason, there is a threshold for the number of value columns (which you can change).
  • Once you return to the other tabs, they are updated accordingly

Data format

The easiest way to learn how to format your data for the DataExplorer is to download the Excel example data. Excel workbooks (.xlsx, .xls) are the recommended file type. Output of Synavision's Digitaler Prüfstand is supported as well. Generic comma-separated value (csv) files are not guaranteed to work, because the separator (comma, semicolon, whitespace) has to be guessed.

The format for Excel files is very simplistic, but please consider these basic rules:

  • Only the first sheet of the uploaded workbook is read in. The contents of the other sheets are ignored
  • The first row must contain the headers (names) of each column
  • The data must start at the second row
  • Do not include any empty rows within your data
  • Missing values in columns are allowed and will be displayed as NaN (not a number)
  • Excel's time format is automatically recognized and displayed properly. Use it to plot pretty time series

The DataExplorer lets you filter your data by classes:

  • Columns containing text instead of numbers are used as class columns
  • The names of these columns are used as classifications (A classification is a group of several classes)
  • The contents of these columns are used as classes (within each classification)
  • You can force a column to be recognized as a class column, if its header contains an exclamation mark "!". This is useful if your classes are represented by numbers
  • All other columns are used as value columns
  • You do not need classifications, but it makes the DataExplorer more useful
  • If no classifications are found, a classification File with the name of the file as a class is created

Notes on appending new data:

  • For the upload, you can choose between replace and append
  • Append will join all columns with identical header names in the old and new data
  • The file names are automatically used as additional classes in a new classification called File
